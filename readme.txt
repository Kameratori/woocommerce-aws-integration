== Description ==

A WordPress plugin to produce SNS events from WooCommerce hooks.

You can configure this plugin to publish metadata from any SNS Topic triggered
by the following WooCommerce events:

- Order Paid
- Order Shipped
- Order Refunded
- Product Published
- Product Sold
- Product Shipped
- Product Refunded
